# Python String Formatting Examples

A short, beginner-friendly notebook demonstrating Python string formatting with f-strings: basic variable insertion, alignment, and simple format specifiers.


## Overview

This notebook (`08_f-strings.ipynb`) explains and demonstrates Python f-strings (formatted string literals). It is designed for readers who already know the basics of Python and want to learn or practice how to format strings clearly and concisely using f-strings.

Key learning goals:
- Insert variables into strings using f-strings.
- Control alignment and width.
- Use basic format specifiers (examples can be extended to numeric and date/time formats).
